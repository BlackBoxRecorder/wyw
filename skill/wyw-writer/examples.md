# WYW 写作范例

以下为三种典型文档类型的完整 `.wyw` 写作示例。

---

## 示例一：纯散文（带逐段翻译）

适合古文散文、奏表、书信等。

```
---
title: 爱莲说
author: 周敦颐
dynasty: 宋
---

水陆草木之花，可爱者甚[蕃](多)。晋[陶渊明](东晋著名诗人)独爱菊。自[李唐](指唐朝)来，世人甚爱牡丹。
>> 水上、陆地上各种草本木本的花，值得喜爱的非常多。晋代的陶渊明唯独喜爱菊花。从唐朝以来，世间的人们非常喜爱牡丹。

予独爱莲之出[淤](污泥)泥而不染，[{濯|zhuó}](洗涤)清{涟|lián}(水清而有微波的样子)而不[妖](美丽而不端庄)。
>> 我唯独喜爱莲花从淤泥中长出却不被污染，经过清水的洗涤却不显得妖媚。

予谓菊，花之[隐逸](隐居避世)者也；牡丹，花之富贵者也；莲，花之君子者也。
>> 我认为菊花，是花中的隐士；牡丹，是花中的富贵者；莲花，是花中的君子。
```

**关键点：**
- `[词](释义)` 用于注释生僻词
- `{字|拼音}` 用于标注读音
- 每个段落后紧接 `>>` 译文
- 空行分隔不同段落组

## 示例二：诗词围栏块（带注音和注释）

适合诗词作品，支持内部标题和作者信息。

```
---
title: 宋词·念奴娇·赤壁怀古
author: 苏轼
dynasty: 宋
---

::: poetry
# 念奴娇·赤壁怀古
:: 苏轼

大江东去，浪{淘|táo}尽，千古[风流人物](杰出不凡的人物)。
故{垒|lěi}西边，人道是，三国周郎[赤壁](地名，在今湖北赤壁市)。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少{豪|háo}杰。

遥想[{公}{瑾|jǐn}](指周瑜，字公瑾)当年，小乔初嫁了，雄姿英发。
羽扇{纶|guān}巾，谈笑间，[{樯|qiáng}{橹|lǔ}](代指曹操的水军)灰飞烟灭。
故国[神游](在想象中游历)，多情应笑我，早生{华|huā}发。
人生如梦，一尊还[{酹|lèi}](把酒浇在地上，表示祭奠)江月。
:::
```

**关键点：**
- `::: poetry` 包裹整个诗词
- `#` 行作为诗词标题
- `:: 作者` 行作为元信息
- 诗词正文每行独立，支持内联语法
- 空行分隔 verse 段落

## 示例三：内联语法密集使用

展示最复杂的内联语法组合。

```
---
title: 石壕吏
author: 杜甫
dynasty: 唐
---

暮投石壕村，有吏夜捉人。老翁{逾|yú}墙走，老妇出门看。
>> 傍晚投宿石壕村，有差役夜里来强征兵。老翁翻墙逃走，老妇出门去应对。

吏呼一何怒！妇啼一何苦！听妇[前致词](走上前去对差役说话)：三男[{邺|yè}{城}{戍|shù}](三个儿子在邺城服役)。
>> 差役吼得多么愤怒！老妇哭得多么凄苦！听见老妇上前对差役说：三个儿子都去邺城服役了。

*一男*附书至，*二男*新战死。[存者](活着的人)且[偷生](苟且活着)，[死者](死去的人)长已矣！
>> 一个儿子捎信回来，两个儿子最近战死。活着的人暂且偷生，死去的人就永远完了！
```

**关键点：**
- `[{邺|yè}{城}{戍|shù}](释义)`：三字词组，其中两个字有注音，整词有注释
- `[前致词](释义)`：注释可直接接在词后面
- `*一男*`：强调标记可与其他内联语法混合
- 段落与译文一一对应

## 示例四：完整古籍文章（带分隔线和引用）

展示 Frontmatter 标题、分隔线、引用块等块级语法的组合。

```
---
title: 岳阳楼记
author: 范仲淹
dynasty: 宋
---

庆历四年春，[滕子京](范仲淹友人，名宗谅){谪|zhé}守巴陵郡。越[明年](第二年)，政通人和，百废[具兴](同"俱兴")，乃重修岳阳楼。
>> 庆历四年的春天，滕子京被贬官到岳州做知州。到了第二年，政事顺利，百姓和乐，各种荒废的事业都兴办起来了。

---

予观夫巴陵[胜状](美好景色)，在洞庭一湖。衔远山，吞长江，浩浩{汤|shāng}汤，横无际涯。
>> 我看那巴陵郡的美好景色，全在洞庭湖上。它连接着远处的群山，吞吐着长江的流水，水波浩荡，广阔无边。

---

> 先天下之忧而忧，后天下之乐而乐。

```

**关键点：**
- `---` 在正文中作为分隔线（非 frontmatter）
- `>` 引用块用于名言引用
- `[词](释义){字|拼音}` 可以连用（注释后紧跟注音标记，两者独立解析）
