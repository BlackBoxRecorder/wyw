# 文言文标记语法使用说明

本文档介绍 `.wyw` 文件的标记语法，用于编写带注释、注音和现代文翻译的文言文。

---

## 文件结构

`.wyw` 文件由两部分组成：
1. **Frontmatter** - 文件顶部的元数据（可选）
2. **正文内容** - 实际的文言文及标记

---

## 一、Frontmatter（元数据）

文件开头使用 YAML 格式定义文档元数据，包裹在 `---` 之间：

```wyw
---
title: 陋室铭
author: 刘禹锡
dynasty: 唐
---
```

### 支持的字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 文章标题 | `陋室铭` |
| `author` | 作者姓名 | `刘禹锡` |
| `dynasty` | 所属朝代 | `唐` |

---

## 二、块级语法

### 1. 标题

使用 `#` 表示标题，支持 1-3 级：

```wyw
# 一级标题
## 二级标题
### 三级标题
```

### 2. 段落

普通文本行自动识别为段落。段落之间用空行分隔：

```wyw
这是第一段文言文。

这是第二段文言文。
```

### 3. 译文

使用 `>>` 标记现代文翻译，紧跟在对应文言文段落后：

```wyw
山不在高，有仙则名。

>> 山不在于有多高，有了仙人居住就会出名。
```

**特点：**
- 译文会自动与上方段落关联
- 支持多行译文（每行都以 `>>` 开头）

### 4. 引用块

使用 `>` 标记引用内容：

```wyw
> 孔子曰：学而时习之，不亦说乎？
```

### 5. 分隔线

使用三个或以上连字符表示分隔线：

```wyw
---
```

### 6. 校对日期

使用 `--YYYY年M月D日--` 标记校对日期，生成页脚：

```wyw
--2024年1月15日--
```

### 7. 诗词围栏块

使用 `:::` 包裹诗词内容，支持内部标题和元信息：

```wyw
::: poetry
# 静夜思
:: [唐]李白

床前明月光，
疑是地上霜。

举头望明月，
低头思故乡。
:::
```

**说明：**
- `::: poetry` - 开始围栏块（类型默认为 poetry）
- `# 静夜思` - 诗词标题（可选）
- `:: [唐]李白` - 元信息行（作者、朝代等，可选）
- 诗词内容中支持所有内联语法（注音、注释等）
- 空行用于分隔诗句段落
- `:::` - 结束围栏块

---

## 三、内联语法

### 1. 注音（Ruby 标注）

使用 `{字|拼音}` 为汉字添加拼音标注：

```wyw
有{仙|xiān}则名
```

**效果：** 在"仙"字上方显示拼音 "xiān"

### 2. 注释

使用 `[词](释义)` 为词语添加注释：

```wyw
[陋室](简陋的屋子)
```

**效果：** "陋室"显示为带注释的词，释义为"简陋的屋子"

### 3. 注音+注释组合（单字）

当同一个字既有注音又有注释时，使用注释包裹注音：

```wyw
春眠不觉[{晓|xiǎo}](天刚亮的时候)，处处闻啼鸟。
```

**效果：** "晓"字上方有拼音"xiǎo"，同时可悬停查看注释

**语法说明：** `[{字|拼音}](释义)` - 中括号内放置注音标记，小括号内放置注释

### 4. 注音+注释组合（整词）

当注释针对多字词组，且其中部分字需要注音时，在中括号内放置多个大括号块：

```wyw
青[{箬|ruò}{笠}](用箬竹叶或竹篾编成的斗笠)，绿[{蓑|suō}{衣}](用草或棕毛编织的防雨用具)。
```

**效果：**
- "箬"字上方显示拼音"ruò"
- "箬笠"整体显示为带注释的词组，鼠标悬停"箬笠"显示完整注释

**语法说明：** `[{字|拼音}{字}...](释义)`
- 中括号内每个字用 `{}` 包裹
- 需要注音的字使用 `{字|拼音}` 格式
- 不需要注音的字使用 `{字}` 格式
- 小括号中为整词的注释内容

**更复杂的示例：**

```wyw
三男[{邺|ye}{城}{戍|shù}](三个儿子在邺城服役。邺城：即相州，在今河南安阳)。
```

### 5. 着重（强调）

使用 `*文本*` 标记需要强调的内容：

```wyw
*孔子*云：何陋之有？
```

## 四、完整示例

```wyw
---
title: 陋室铭
author: 刘禹锡
dynasty: 唐
---

# 陋室铭

## 全唐文

### 刘禹锡

---

*山不在高，有仙则名*。水不在深，有龙则灵。[{斯|sī}{是}{陋|lòu}{室}](这是简陋的屋子)，惟吾[{德}{馨|xīn}](香气，这里指品德高尚)。
>> 山不在于有多高，有了仙人居住就会出名。水不在于有多深，有了龙的存在就会有灵气。这是一间简陋的屋子，只因我的品德高尚就不感到简陋了。

[{苔|tái}{痕|hén}](苔藓碧绿)[上阶](长到，蔓到台阶)绿，草色入帘青。谈笑有[鸿儒](博学的人)，往来无[白丁](没有功名、学问的平民)。可以调素琴，阅[金经](用泥金书写的佛经)。无[丝竹](弦乐器和管乐器，泛指世俗的音乐)之乱耳，无案{牍|dú}之劳形。
>> 苔藓的痕迹蔓延到台阶上，呈现出一片碧绿；草色映入竹帘，使室内染上青色。在这里谈笑的都是渊博的大学者，往来的没有知识浅薄的人。可以弹奏不加装饰的古琴，阅读用泥金书写的佛经。没有嘈杂的音乐扰乱听觉，没有繁重的官府公文劳累身体。

[南阳](今河南南阳)诸葛[庐](草屋)，[西蜀](今四川成都一带)[子云](汉代文学家扬雄的字)亭。孔子云：[何陋之有](有什么简陋的呢？语出《论语·子罕》)？
>> 它就像南阳诸葛亮的草庐，西蜀扬子云的玄亭。正如孔子所说：这有什么简陋的呢？

> 陋室铭 - 刘禹锡
```

## 五、语法速查表

| 语法 | 用途 | 示例 |
|------|------|------|
| `---` | Frontmatter 分隔 / 分隔线 | `---\ntitle: 标题\n---` |
| `#` `##` `###` | 标题 | `# 一级标题` |
| `>>` | 译文 | `>> 现代文翻译` |
| `>` | 引用 | `> 引用内容` |
| `---` | 分隔线 | `---` |
| `--YYYY年M月D日--` | 校对日期 | `--2024年1月15日--` |
| `:::` | 围栏块 | `::: poetry\n内容\n:::` |
| `:: text` | 围栏元信息 | `:: [唐]李白` |
| `{字\|拼音}` | 注音 | `{仙\|xiān}` |
| `[词](释义)` | 注释 | `[陋室](简陋的屋子)` |
| `[{字\|拼音}](释义)` | 注音+注释（单字） | `[{晓\|xiǎo}](天刚亮)` |
| `[{字\|拼音}{字}...](释义)` | 注音+注释（整词） | `[{箬\|ruò}{笠}](斗笠)` |
| `*文本*` | 着重 | `*强调*` |

---

## 六、注意事项

1. **空行分隔**：段落之间需要空行分隔，否则会被合并为一个段落
2. **译文关联**：译文块（`>>`）会自动与紧邻的上方段落关联
3. **嵌套支持**：着重标记 `*` 支持内部嵌套其他内联语法
4. **转义**：目前暂不支持转义，避免在文本中直接使用特殊标记字符
